
# Real-Time Tracking System

A simple yet powerful **real-time tracking system** built using **Vanilla JavaScript**, **Embedded JavaScript (EJS)**, and **CSS3**.  
This application allows **two devices to connect and track each other’s location simultaneously**, providing an interactive and dynamic experience.

---

## Features

- **Two-Device Real-Time Connection** – Connect and track between two devices seamlessly.  
- **Lightweight Frontend** – Built entirely with Vanilla JS and EJS for faster rendering and zero external dependencies.  
- **Modern UI** – Styled with clean and responsive CSS3 for a smooth user experience.  
- **Instant Updates** – Location updates are reflected instantly on both devices.

---

## Tech Stack

- **Frontend:** Vanilla JavaScript, EJS, CSS3  
- **Backend:** Node.js (with Express, if applicable)  
- **Real-Time Communication:** Socket.io (if used)  
- **Rendering Engine:** Embedded JavaScript (EJS)

---

## How It Works

1. Open the tracker app on **two different devices**.  
2. Each device generates or shares a **unique connection ID**.  
3. Once connected, both devices can **see each other's live location** in real-time.  
4. The map or coordinates update automatically without refreshing the page.

## ⚡ Installation

1. Clone the repository:

git clone https://github.com/Sohum-Nikam/RealTime-Tracker.git


2.	Navigate to the project folder:


cd RealTime-Tracker

3.	Install dependencies (if using Node.js backend):

npm install


4.	Run the application:

npm start

or

node app.js


5.	Open the app in your browser:

http://localhost:3000
